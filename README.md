# JSONPlaceholder App

A news-like app using api from https://jsonplaceholder.typicode.com/

## Installation

1. After cloning the project, open your terminal and run this line 
```
pod install
```

2. Open the .xcworkspace file

### Built with
- **Swift 5**
- **Swift MVVM**
- **RxSwift**
- **Snapkit**
- **LightBox**
